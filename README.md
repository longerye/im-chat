# im-chat

#### 介绍
聊天IM，基于Netty技术仿微信pc项目，支持单聊、群聊文字、图片、实时音视频通话等功能

#### 软件架构
软件架构说明
系统采用前后端分离架构模式，前端项目使用Vue3开发， 后端项目主要使用SpringBoot+Netty+Redis+Minio+第三方服务等技术栈实现
